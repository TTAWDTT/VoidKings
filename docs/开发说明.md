## 首次运行配置

⚠️ **重要：cocos2d-x 引擎需要单独配置！**

由于 cocos2d-x 引擎体积较大，已添加到 `.gitignore` 中，克隆仓库后需要手动配置：

1. **获取 cocos2d-x 引擎**
   - 从项目组其他成员处获取已配置好的 `cocos2d` 目录（推荐）
   - 或从 [cocos2d-x GitHub](https://github.com/cocos2d/cocos2d-x/releases) 下载 v4.0 版本

2. **放置到项目根目录**
   - 将 `cocos2d` 目录放到项目根目录下（与 `Classes`、`proj.win32` 同级）
   - 确保目录结构正确：`VoidKings/cocos2d/cocos/2d/...`

3. **打开解决方案**
   - 用 Visual Studio 打开 `proj.win32/VoidKings.sln`
   - 右键解决方案 → "重定解决方案目标" → 选择你的 VS 版本
   - 生成解决方案

如果遇到问题，请查阅 `docs/故障排除指南.md`。
