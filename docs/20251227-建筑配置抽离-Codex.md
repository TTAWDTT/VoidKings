# 20251227 建筑配置抽离 / Codex

## 改动概览
- 修复结算页进入下一关时兵种不移动的问题（旧场景清空静态敌对列表导致失效）。
- 建筑配置与商店选项抽离到 `Resources/res/buildings_config.json`，统一由 `BuildingManager` 解析加载。

- 配置集中后更便于调参和复用，减少多处硬编码不一致风险。
