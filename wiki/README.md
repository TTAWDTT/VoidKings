## 📝 使用说明

1. **阅读代码**：每章都包含大量源码片段，建议对照实际代码阅读
2. **动手实践**：每章末尾都有练习题，建议完成以加深理解
3. **逐步深入**：从基础开始，按顺序学习效果最佳
4. **参考附录**：遇到问题时查阅附录中的规范和 FAQ
