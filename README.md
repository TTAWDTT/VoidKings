# VoidKings
一个基于 Cocos2d-x 开发的类“部落冲突”策略游戏项目，为 2025 年下半学期同济大学软件工程（精英班）课程大作业。项目成员：孟圣雨、罗臻、尹佳玮。

## 游戏规则

### 游戏目标
建设你的基地，训练部队，进攻敌人基地以获取资源！

### 建筑类型
- **大本营**：核心建筑，保护你的基地
- **加农炮/箭塔**：防御建筑，攻击敌人
- **金矿/圣水收集器**：随时间产出资源
- **金库/圣水库**：存储你的资源
- **兵营**：训练部队用于战斗
- **城墙**：阻挡敌方部队

### 战斗系统
- 在敌方基地边缘部署部队
- 摧毁50%获得1星，摧毁大本营获得2星，100%摧毁获得3星
- 通过摧毁敌方建筑获得金币和圣水
- 战斗时间限制：3分钟

## 目录结构概要
```

Resources/      资源（纹理、字体、音效、配置）
cocos2d/        引擎源码（通过模板复制） - 勿修改
docs/           项目与参考文档
proj.win32/     VS 原生工程（如需手工打开）- 我们的第二个工作区
out/            构建输出（按架构/配置划分） - 勿修改
```
